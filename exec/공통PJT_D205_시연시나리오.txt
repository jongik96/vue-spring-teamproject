1. 회원가입 및 로그인
처음 접속하면 메인화면으로 타임라인화면이 나타납니다.
우측상단의 'Sign Up' 버튼을 눌러 회원가입화면으로 이동합니다.
'Email', '비밀번호' ,'비밀번호확인', '닉네임' 입력칸은 조건에 맞게 입력이 완료되어야 합니다.
'선호 언어' 와 '선호 알고리즘'은 빈칸으로 제출해도 무방합니다
(회원가입 이후 회원정보 수정에서 추가 가능합니다)

회원가입이 정상적으로 완료되면 로그인 창으로 넘어가며 방금 가입했던 계정으로 바로 로그인 하실 수 있습니다.
처음 로그인을 하게 되면 현재 팔로우하고있는 유저, 문제, 알고리즘 등이 없는 상태이기 때문에 타임라인에는 아무런 글이 뜨지 않는 것이 정상입니다.
우측 상단의 메뉴들을 이용해 왼쪽부터 차례대로 타임라인, QnA타임라인, 토론게시판, 알림창, 프로필페이지로 이동할 수 있습니다.
중앙 상단의 '검색해주세요' 창을 클릭하면 검색 모달창이 나타나게 됩니다.

2. 검색
검색 카테고리로 문제,알고리즘,유저 중 선택할 수 있습니다.
문제를 선택한 경우에는 검색하고 싶은 문제 사이트, 문제 번호, 필요에 따라 해당 언어를 선택하고 알고리즘을 해시태그에 포함시켜 검색할 수 있습니다.
( 문제사이트와 문제번호를 정하지 않고 검색하실 수 없습니다. )
알고리즘을 선택한 경우에는 해당 언어와 알고리즘 두 가지를 선택해 검색 하실 수 있습니다.
( 알고리즘 검색의 경우 특정 알고리즘을 반드시 1개 이상 선택해야 검색이 가능합니다. )
유저 검색의 경우 만약 'a'를 입력한다면 a로 시작하는 닉네임을 가진 유저들을 한 번에 검색 가능합니다.

3. 팔로우
2. 검색을 통해 유저, 알고리즘, 문제 등을 팔로우 하는것이 가능합니다. 문제를 팔로우 한 뒤 다시 타임라인으로 돌아가면 타임라인에 자신이 팔로우한 유형의 게시글들이 뜨는 것을 볼 수 있습니다.
이후에 다시 검색을 통해 팔로우를 취소하거나 자신의 프로필페이지를 통해 팔로우리스트페이지에서 팔로잉을 취소하고 난 뒤 다시 타임라인으로 돌아가면 팔로우를 취소했기 때문에 해당 글들이 다시 타임라인에 나타나지 않는 것을 볼 수 있습니다.

4. 풀이 요청
특정 유저의 프로필페이지로 들어가면 프로필이미지 바로 밑에 문제풀이요청 버튼이 있습니다. 
문제풀이요청 버튼을 클릭하면 문제풀이요청 게시글 작성 화면으로 이동하게 됩니다.
문제 풀이 요청 시 문제사이트와 문제번호, 질문 내용을 반드시 입력해야 합니다.
글을 작성한 뒤 제출하기 버튼을 누르면 문제풀이요청을 보낸 회원의 프로필페이지로 다시 이동하게 됩니다.
프로필 하단의 요청받은 문제 탭을 클릭하시면 방금 요청한 글과 글의 답변 상태를 확인하실 수 있습니다.
(자신의 프로필페이지에서도 '내가 요청한 문제' 탭에서 풀이를 요청한 글과 답변상태를 확인 가능합니다.)

5. 게시글 작성
타임라인에서 타임라인의 가장 위의 '어떤 문제를 푸셨나요?' 입력창을 클릭하면 게시글 작성페이지로 이동하게 됩니다.
카테고리는 '문제풀이'가 기본으로 선택되어있으며 변경하실 수 있습니다. 
자신의 풀이에 해당하는 알고리즘, 문제사이트, 문제번호와 사용언어, 제목을 입력한뒤 내용을 작성하고 '작성하기' 버튼울 누르면 게시글 작성이 완료됩니다.
(내용 작성 시에 마크다운 언어 사용이 가능합니다.)

6. 풀이 요청 답변
우측 상단의 프로필이미지를 클릭하면 '정보수정','프로필페이지','로그아웃' 버튼을 볼 수 있습니다.
프로필페이지를 선택해 자신의 프로필 페이지로 들어간 다음 프로필 하단의 '요청받은 문제' 탭을 클릭하면 지금까지 요청 받았던 문제들을 확인할 수 있습니다.
요청받은 문제에서 수락하기 버튼을 누르게 되면 답변이 가능하며 한 번 거절한 문제는 다시 되돌릴 수 없습니다.
수락하기 버튼을 누른 뒤 답변하기 버튼을 누르면 게시글 작성 페이지로 이동이 됩니다. 
이후 4. 게시글 작성 의 방법과 동일합니다.

7. 프로필페이지
자신의 프로필 페이지에서는 프로필이미지 하단의 'select image'버튼을 클릭해 이미지업로드 화면으로 이동한 뒤 원하는 프로필이미지를 업로드 할 수 있습니다.
'팔로워', '팔로잉' 텍스트를 클릭하면 팔로워 팔로잉 정보를 나타내는 페이지로 이동하게 되며
해당 페이지에서 특정 유저,알고리즘,문제의 팔로잉취소가 가능합니다. 또한 다른 유저의 이름을 클릭해 해당 유저의 프로필페이지로 이동할 수 있습니다.
연필모양 버튼을 클릭해 한줄소개를 설정하실 수 있습니다.

프로필하단에서는 탭 별로 게시글, 내가 요청한 문제, 요청받은 문제들을 확인하실 수 있습니다.

8. 토론게시판
토론게시판은 특정 대회가 종료되었을 시에, 관리자가 직접 해당 대회의 문제별로 실시간댓글소통이 가능한 토론방을 만들어 주는 기능을 제공합니다.
예시로 'KAKAO 2021 CODE 1번' 토론방을 클릭하게 되면 해당 토론방으로 이동하게 되며, 댓글에서 마크다운언어를 이용한 원활한 의견 공유가 가능합니다.

9. 실시간알림창
내가 쓴 글에 댓글, 좋아요 이벤트가 발생했을 때, 나를 팔로우하거나 나의 문제풀이요청에 대한 답변글이 생겼을 때 활성화 되며 
우측 상단의 알림 모양버튼을 클릭하면 확인할 수 있습니다.
